# Final Prompt

生成一张竖版封面图（约 3:4，安全边距 6–8%，不要裁切标题/主视觉）。
固定系列版式：顶部标题；右下放置一本发光的资源目录悬浮卡（不要出现任何真人/人脸/人物）；主题主视觉集中在左下与中下区域；背景纹理随主题变化。

【标题（固定风格 + 关键词高亮 + 轻弧排版）】
- 主标题整体风格：粗体、深色粗描边、立体阴影；缩略图清晰可读
- 文案拆行：
  1）给 AI 用的（明黄 #ffe566，粗黑描边）
  2）设计灵感库（电光橙 #ff6b35，更粗描边 + 轻外发光，关键词高亮）
- 关键词高亮策略：仅「设计灵感库」用高亮橙 + 更粗描边 + 轻外发光
- 轻弧排版策略：「设计灵感库」整行沿很轻微弧线排布（左下→右上微上扬），不可夸张
- 识别锚点：小标签条「Design Resource」

【人物位（右下角）— 无真人】
- 不要出现人、人脸、手、身体剪影
- 用一本厚重的深色资源目录书/卡片替代：封面印着 INS / MOT / CMP / SKL，边缘品红紫发光

【主题主视觉（贴题）】
- 封面类型：概念总览
- HERO 场景隐喻：3–4 个悬浮浏览器窗口叠放，显示现代设计网站界面（暗色 UI、渐变背景、网格布局），窗口有细微发光描边，像被吸进右侧资源目录
- 关键点呈现：底部微弧排布小标签「灵感站」「动效」「组件」「Skills」
- 视觉风格：高对比、粗描边、发光边缘；信息层级：标题 > HERO 窗口 > 目录卡 > 标签 > 背景

【主题配色与背景】
- 主题配色：主色深紫黑 / 辅色品红紫 / 高亮电光橙
- 背景基底纹理：暗色科技网格 + 细粒子 + 轻微 HUD 线，对比压低不抢标题

【总体】
- 缩略图优先：减少细碎小字，关键元素要大
- 四周留白，平台裁切也不影响主体可读
- 无人物、无水印、无二维码

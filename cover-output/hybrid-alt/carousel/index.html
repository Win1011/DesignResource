04 / Modes

有没有参考
都行

把库丢给它
两种都稳

Mode A · 有参考

有想法、有意向

把参考 / 意向 + 这份库一起丢给 AI。它对方向、选资源，少跑偏，落地更稳。

Mode B · 没参考

没有参考也行

只丢需求 + 这份库。它按分类自己选型，你可以放心交给它生成。

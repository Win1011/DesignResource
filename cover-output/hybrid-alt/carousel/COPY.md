# 小红书组图定稿文案（硬切分栏 · Agent 视角）

对应库版本：Design Resource v3.1 · 31 条 · 五大类

定位：给每个用户的 AI 用的设计选型目录（不是给人翻的收藏夹）

## 01 封面
- 主标题：AI 的设计灵感库
- 副文案：交给你的 AI，让它自己会选型
- 信息点：30+ for agents

## 02 这是什么
- 主标题：给 Agent 读的设计目录
- 1. 专为 AI 工作流整理
- 2. 类型分开放
- 3. 写清怎么用

## 03 库里有什么
- 主标题：五大类，AI 按需取
- INS / CMP / MOT / DEV / SYS

## 04 有没有参考都行
- Mode A 有参考：想法 + 库 → 对方向、选资源、少跑偏
- Mode B 没参考：需求 + 库 → 自己选型，放心生成

## 05 什么时候丢给 AI
- 不知道网页方向 → INS
- 缺组件 / 动效 → CMP / MOT
- 要落地动画或规范 → DEV / SYS

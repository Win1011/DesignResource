04 / Workflow For Agents

AI 怎么用

三步选型
不要一把梭

01

按目标进分类 先判断是找灵感、找组件、做动效，还是装 Skill

02

每次只推 1–3 个 说明为什么选它、输入输出是什么、要不要写代码

03

用稳定 ID 引用 例如 INS-001、MOT-001、SKL-005，方便后续自动化

Example

做营销页动效 → 先看 INS / CMP，实现用 DEV-001 Motion，需要社交封面再调 SKL-005。

# 反廉价黑名单（完整版）

> 哲学（借鉴 finesse，加我们的改进）：大多数 AI 味是*模型不假思索就够的默认*。修法几乎从不是"加更多"，而是"命名这个反射，拒绝它，落到一个有意图的选择"。这份清单既是负面禁区，也带正向替代——光避开 tell 仍可能平庸，物理层配方（见 SKILL.md「高级感物理层」）才是正向生成的底座。

---

## 0. 两层反射检查（先跑这个）

命名品类默认还不够——避开第一层，第二层默认在等你。两层都打到"猜不出"。

- **一阶：** 光从品类能猜出配色/风格？能→训练数据默认，重做。（咖啡→米色黄铜；AI/SaaS→紫光Inter；金融→海军金；极简→编辑衬线大字）
- **二阶：** 避开一阶后，从"品类+反例"还能猜出？("不奶油的 AI 工具"→编辑排版；"不海军金的金融"→终端暗色；"不米色的手作"→牛血+Cormorant)→这是更深的陷阱，精心做的页常死在这层。

当前二阶饱和赛道（要用就得说出理由）：编辑排版风（衬线+米色+巨字）、磷光终端暗色、粗野 Helvetica 骨色、AI 紫光。无理由禁用。

## 1. `[硬禁]`（命中一条不交付，无例外）

- **em-dash 当文案装饰**（— 做停顿/列项/戏剧旁白）。AI 文案最常犯的 tell。用句号、逗号或重写句子。禁的是可见文案，不是代码。
- **div 造假截图/假仪表盘/假终端**——拿 `<div>` 摆拍一个"产品预览"冒充页面在描述的真产品。→ 真截图、生成图、或做个真组件。
  - 边界（不算违规）：mock 是产品自己的功能（CMS 预览窗、设计工具画布、图片热点标注、带波形聊天气泡）——它代表 UI 真实拥有的状态，不是摆拍替代缺位的截图。且必须由真实几何/数据驱动、能交互、不带假数字。
- **渐变文字 `background-clip:text` 当默认 flourish**。→ 单色，或字重/字号强调。（例外：双 accent 锁色的刻意 duotone 灵魂动作）
- **AI 紫光/`#6366f1`**——紫渐变按钮、霓虹 mesh 背景、发光 CTA 当默认。→ 中性底+一个高对比刻意选的 accent。（例外：brief 明确要紫，且执行有意图）
- **伪造精确数字**——`92%`/`4.1×`/`48k`/`5.8mm` 凭空造来凑规格感。→ 用真数据、标 mock、或删数字。

## 2. AI 味（reject as default；override 需说明理由）

**布局与节奏**
- **每节 eyebrow 小标**——标题上方那个大写宽字距小标签（ABOUT/OUR PROCESS），AI 输出里 55–95% 都有。→ ≤1/3 节才许有。机械检查：数 `uppercase tracking` 小标，> `ceil(节数/3)` 即 fail。
- **01·02·03 编号当默认架构**。→ 只用于真序列（真三步流程、时间线）。
- **千篇一律卡片网格**——6 张图标+标题+段落同尺寸。→ 变 tile 尺寸、`auto-fit minmax(280px,1fr)`、交替全宽行、给 2–3 格真视觉变化。
- **布局族重复**——同一种布局出现两次以上。→ 一种布局族至多 2 次；8 节页 ≥4 种布局族；image+text 锯齿至多连续 2 次。
- **分裂式标题**——左大标题+右小说明段当章节头。→ 垂直堆叠一个焦点；右栏只放真视觉时才分。

**视觉与 CSS**
- **侧条纹边** `border-left:3px` accent。→ 全边、bg 染色、引导图标、或不要。
- **玻璃拟态当默认装饰**。→ 罕见且有目的；用就加 1px 内边框+内阴影做真折射，带 `prefers-reduced-transparency` 实色兜底。
- **Hero=文字+渐变 blob**。blob 不是 hero 视觉。→ 真引擎或真图。
- **纯 `#fff`/`#000`、硬 `#333` 边、亮底纯黑投影**。→ 染色中性、半透明边、色相匹配阴影。
- **圆角系统混乱**——圆按钮配方布局。→ 选一个圆角刻度锁死。
- **按钮对比失败**——白按钮白字、透明按钮融进背景、照片上幽灵按钮无衬底。→ 每个 CTA label vs bg ≥4.5:1（18px+粗体 ≥3:1）；照片上幽灵按钮加 scrim/blur/1px 描边。
- **表单对比失败**——近白占位符在近白输入框、白卡浮在白节上。→ 每个输入/占位/聚焦环/帮助文/错误文都对它所在节测对比，不是对页面默认。

**字体**
- **盲用反射默认字体**：`Inter, Fraunces, Instrument Serif, Playfair, Cormorant, Space Grotesk, Syne, DM Sans, Newsreader, Lora`。都是好字体也都用烂了。→ 带理由可用；否则换不那么疲劳的 sans-display（Geist / Cabinet Grotesk / PP Neue Montreal / GT Walsheim / ABC Diatype）。
- **"创意=衬线"反射**。→ 默认 sans-display；衬线只在真编辑/奢侈/传承且能说出为什么这衬线配这品牌时用。
- **混族强调**——sans 标题里塞衬线词。→ 同族斜体/加粗。
- **mono 当"技术感"偷懒速记**、全大写正文。→ 罕见且有意。

**配色**
- **米色+黄铜+咖啡**当手作/传承默认。2026 饱和默认。禁用色族：
  - bg: `#f5f1ea #f7f5f1 #fbf8f1 #efeae0 #ece6db #faf7f1 #e8dfcb`
  - accent: `#b08947 #b6553a #9a2436 #9c6e2a #bc7c3a #7d5621`
  - text: `#1a1714 #1a1814 #1b1814`
  - token 名 `--cream --sand --bone --paper --flour --linen` 本身就是 tell。
  → 轮换：冷奢（银灰+铬+烟）、森林（深绿+骨+琥珀）、黑棕（真近黑+暖棕无米色）、钴蓝+奶油、赤陶+板岩、橄榄+砖+纸、纯单色+一个饱和 pop。连续两个手作 brief 不许同色族。
- **THE LILA RULE——AI 紫光**及通用 indigo 表亲。tech/SaaS/AI 默认禁：紫蓝渐变按钮、霓虹 mesh 背景、近黑上去饱和 indigo（`#5E6AD2` Linear 克隆）、`#6366f1`。
  → 中性底（Zinc/Slate/Stone 或染色近黑）+ 一个高对比刻意 accent。（覆盖：brief 明确要紫且执行有意图。）
- **其他品类默认色族**——命名并拒绝再设计：金融→海军金；健康护肤→鼠尾草绿+奶油；金融仪表盘→`#020617`+通用绿红无品牌色相。品类能预测配色就重做（见 §0）。
- **accent 漂移**——暖灰站第 7 节冒蓝 CTA。→ 配色锁定。
- **滚动中途反转主题**——暗页里塞暖纸节。→ 一个主题锁死（除非刻意一次性滚动换主题）。
- **饱和爆表**——accent chroma 推到霓虹廉价。→ 主饱和默认克制（OKLCH chroma ≤0.23；亮度高再压 ≤0.18）。

**内容与文案**
- **eyebrow 堆砌+废话文案**；卖萌错位双关；假匠人标签；假诗意微元数据。→ 重读每个可见串，换成朴素功能句。
- **营销废话词族**：streamline · empower · supercharge · leverage · unleash · transform · seamless · world-class · enterprise-grade · next-generation · cutting-edge · game-changer · mission-critical · elevate · unlock · revolutionize。→ 挑具体名词+动词说产品*字面上做什么*（"导入你的 Figma 文件"胜过"无缝赋能你的工作流"）。
- **格言节奏腔**——"严肃陈述+短否定"反复（"它不是工具。它是系统。"/"更少噪音。更多信号。"）。一处是钩子，三处以上是 AI tell。
- **不说明动作的按钮/链接**——"OK""提交""点这里""了解更多"×5。→ 动词+对象（"保存更改""删除项目"）；链接独立成义（"查看定价"非"这里"）。
- **"Jane Doe"占位数据**、lorem ipsum。→ 真实具体内容。
- **20 行规格表每行一条发丝线**。→ 分 2–3 组、每规格一卡、滚动 snap pill、"前 5+查看全部"。
- **>3 行引用**、署名只有名字（`— Sarah`）。→ ≤3 行，名+角色+公司，真排印引号。
- **重复 CTA 意图**——一页"联系我们""联系""聊聊"。→ 一个意图一个标签。

**图片**
- **图片暗示的 brief 却零图**（食物/酒店/时尚/旅行/产品）。那是 bug 不是极简。→ 真/生图摄影；克制编辑也至少 2–3 张真图。没素材见 asset-sourcing 决策（生图/图库/占位）。
- **手搓装饰性 SVG 插画当默认**。→ 图标用图标库；品牌标用简单字母组合。
- **纯文字 wordmark 当"信赖墙"**。→ 真 SVG logo（Simple Icons `cdn.simpleicons.org/{slug}`）或生图字母组合。只放 logo，下面别加品类标签。

## 3. 动效 tell

- **声称的动效没兑现**——L3 但静态页。→ 交能跑的动效或降级。
- **无动机动画**——GSAP 到处用因为能用。→ 每个动画一句话说理由（层级/叙事/反馈/状态），否则砍。
- **>1 个跑马灯/页**。→ 选一个它服务的节，其他换布局。
- **`window.scroll` 每帧驱动动画/React state**。→ ScrollTrigger / IntersectionObserver / useScroll / CSS `animation-timeline: view()`。
- **`power1.out`/线性缓动**做揭示（太快=廉价）。→ `power3.out` / `cubic-bezier(.16,1,.3,1)` / spring。
- **无 `prefers-reduced-motion` 兜底**。→ 强制；冻静止帧。
- **引擎卡顿**——粒子太多、FBO 分辨率太高、ScrollTrigger 截断。→ 中端机测；<50fps 简化。

## 4. 30 秒自检（交付前诚实回答）

1. 默认命名并打败了没？（两层反射都过）
2. 一个 accent 锁全页了没？
3. eyebrow 数 ≤ `ceil(节数/3)`？
4. ≥4 种布局族？无一种超 2 次？
5. 该有图的地方有图没？
6. 声称的丰富度真兑现了？60fps？reduced-motion 兜底？
7. 文案干净？无 em-dash、无假数字、无 AI 卖萌？
8. 对比度 AA？按钮/占位/聚焦环都算上？

任何一个"否"都是未交付作品。
